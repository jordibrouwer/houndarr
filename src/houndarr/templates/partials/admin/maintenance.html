Maintenance

Reset all instance settings

Reverts policy settings (batch, cadence, cooldowns, cutoff, upgrade) to defaults. Connections, API keys, and enabled state are kept.

Clear all logs

Empties the Activity log on disk. A single audit breadcrumb stays so you know when the wipe happened.
